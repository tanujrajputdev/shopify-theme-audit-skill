# Product Overview

## What this product is

A **Claude Code skill** — a set of markdown instruction files that teach Claude Code to audit Shopify themes the way a senior Shopify developer would.

Not a SaaS, not an API, not a subscription. It runs entirely inside Claude Code using the user's existing Anthropic subscription. No additional services required.

## Current version

**v2.0** — released 2026

## The core promise

Drop eight files into a Shopify theme project. Ask Claude Code to audit the theme. Get a scored report covering performance, accessibility, conversion, third-party app overhead, SEO, AEO (ChatGPT / Claude / Perplexity citations), and GEO (AI Overviews) — with exact file references, line numbers, and working code fixes — in under five minutes.

## What a user gets from one audit

- A score out of 100 with a letter grade (or two split scores: Technical/CRO and SEO/AEO/GEO)
- Every issue found, with the exact file name and line number
- Working Liquid code fixes for each issue (not just descriptions)
- A prioritized fix order with effort and impact estimates
- Estimated Lighthouse, Core Web Vitals, and search-visibility improvements after fixes
- A "What This Theme Does Well" section so the report is balanced, not just problems

## What makes this different from a manual audit

A senior developer audit of similar scope takes 4–8 hours and costs $500–$2,000. This produces a comparable output in under five minutes. The difference is completeness and consistency — a human may skip Low items under time pressure; this checks every item on the list every time.

## The three search layers this covers (why AEO and GEO exist)

Search has split into three layers:

1. **Classic SEO** — Google, Bing, DuckDuckGo. Still the largest traffic source for most stores.
2. **AEO (Answer Engine Optimization)** — ChatGPT, Claude, Perplexity, Gemini answer product-research questions directly. The user never sees a results page. Invisible stores get zero citations at the moment of purchase decision.
3. **GEO (Generative Engine Optimization)** — Google AI Overviews, Bing Copilot, LLM-powered shopping assistants. Citation depends on schema completeness, crawlability for AI bots, and machine-readable trust signals.

This skill audits all three, separately scored in split-score mode.

## Compatibility

- Works with any Shopify theme: Online Store 1.0 and 2.0
- Works fully offline against local theme files
- Works with the live Shopify MCP for in-store data (live store renders, product data, shop info)
- Best results: both local files AND MCP connected simultaneously
