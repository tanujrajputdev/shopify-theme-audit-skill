## How this product is positioned

**Category:** Claude Code skill / AI-assisted developer tool

**Tagline:** Drop eight files into your Shopify project. Get a scored audit in five minutes.

**Not a competitor to:**
- PageSpeed Insights / Lighthouse (those measure runtime; this measures source code)
- Shopify's own theme inspector (that's limited to Dawn and basic checks)
- Manual developer audits (this extends and speeds them up, does not replace developer judgment)

**Competitive moat:**
- AEO and GEO coverage is unique — no other Shopify audit tool checks AI crawler access, FAQPage schema gaps, or llms.txt
- Output includes working code fixes, not just problem descriptions
- Runs inside Claude Code with zero additional tools, APIs, or subscriptions
- Score + grade makes the report immediately shareable with non-technical stakeholders
